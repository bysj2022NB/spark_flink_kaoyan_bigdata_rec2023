# 开发技术
前端：vue.js

后端：springboot+mybatis-plus

数据库：mysql

算法(机器学习、深度学习)：协同过滤算法(基于用户、基于物品全部实现)、KNN

爬虫：python、requests、chrome_driver

大数据分析：spark、echarts

# 特色/创新点
结合百度地图api展示大学地理位置信息；

通过KNN等机器学习深度学习算法进行预测；

通过协同过滤算法(基于用户+基于物品全实现)推荐院校；

通过Spark完成大数据大屏显示；

通过Python的requests、chrome_driver爬取数据
